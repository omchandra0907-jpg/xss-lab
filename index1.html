📖 How It Works

XSS happens when user input gets inserted into a page without being sanitized. The browser treats it as real HTML/JavaScript and executes it.

// ❌ VULNERABLE — innerHTML parses HTML tags
element.innerHTML = userInput;

// ✅ SAFE — replaces < and > with harmless entities
input = input.replace(/</g, '&lt;').replace(/>/g, '&gt;');
element.innerHTML = input;

// ✅ ALSO SAFE — textContent never parses HTML
element.textContent = userInput;
